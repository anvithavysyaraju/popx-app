# PopX App

A React JS implementation of the PopX design (4 screens).

## Screens
1. **Landing** — Welcome screen with Create Account + Login buttons
2. **Login** — Email/password login form
3. **Register** — Full registration form with agency radio
4. **Account Settings** — Profile page with avatar and bio

## Tech Stack
- React 18
- Vite
- CSS (no external UI library)

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deploy the `dist/` folder to Netlify.
